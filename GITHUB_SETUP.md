## 🆘 Troubleshooting

### Site not appearing?
- ✅ Check Pages is enabled (settings → pages)
- ✅ Source is set to `main` branch, `docs` folder
- ✅ Wait 1-2 minutes for deployment

### Getting "404 Not Found"?
- ✅ Make sure `docs/index.html` exists
- ✅ Check the URL: `https://guvvala39.github.io/imageprocessor/`

### Want a custom domain?
1. Buy domain (GoDaddy, Namecheap, etc.)
2. Add it to Pages settings
3. Update DNS settings (registrar will guide you)
